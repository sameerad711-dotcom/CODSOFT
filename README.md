# Simple CLI Calculator

A lightweight, command-line interface (CLI) calculator built in Python that performs basic arithmetic operations. This project is ideal for beginners looking to understand basic Python functions, user input handling, and conditional logic.

## Features

- **Addition (`+`)**: Adds two numbers.
- **Subtraction (`-`)**: Subtracts the second number from the first.
- **Multiplication (`*`)**: Multiplies two numbers.
- **Division (`/`)**: Divides the first number by the second (includes safety checks for division by zero).
- **Error Handling**: Gracefully handles non-numeric inputs and prevents crashes.
